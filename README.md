# 📊 Analytics Dashboard

A modern, mobile-first analytics dashboard built with **HTML, CSS, and JavaScript**. The application provides a clean and responsive interface for monitoring key business metrics, tracking performance, and managing daily activities.

## Features

- Mobile-first, responsive design
- Interactive dashboard layout
- Key performance metrics
- Revenue trend visualization
- Conversion funnel
- Recent activity feed
- Quick task management
- Light/Dark mode support
- Clean and modern UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```
├── index.html
├── styles.css
├── script.js
└── README.md
```
